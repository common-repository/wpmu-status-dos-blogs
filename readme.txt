== Description ==

Lista todos os blogs criados, é necessário estar ativado a opção de multisite no Wordpress 3.0+ para que funcione corretamente.

== Installation ==
* Ativar as configurações de Multisite do Wordpress, as informações necessárias estão neste link: [Create A Network](http://codex.wordpress.org/Create_A_Network)
* Faça o upload do plugin para o seu blog e depois basta ativar.
